 My\+SQL provides connectivity for client applications developed in .NET compatible programming languages with My\+SQL Connector/\+NET through a series of packages.

My\+Sql.\+Data is the core package of Connector/\+NET. It is compatible with .NET Framework 4.\+6+ and .NET 6.\+0+ and provides classic My\+SQL protocol and My\+SQL X Dev\+API capabilities.
